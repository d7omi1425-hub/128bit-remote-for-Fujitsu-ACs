# 128bit-remote-for-Fujitsu-ACs
I reversed engineered the message sent by the Fujitsu AC remote using my own built 128 bit sniffer to recreate the remote and host it on my machine then port forward the application to control my AC from anywhere
components: 
-LCD screen 2 rows
-10k OHMs potentiometer 
-IR LED 940 nm
-S8050 Transistor
-220 OHMs resistor
-330 OHMs resistor
